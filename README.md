## üìò Overview

The notebook covers all essential Python operator types with explanations and examples:

| Operator Type | Description |
|----------------|-------------|
| **Arithmetic Operators** | Perform mathematical operations like addition, subtraction, multiplication, division, etc. |
| **Comparison Operators** | Compare two values and return Boolean results. |
| **Logical Operators** | Combine conditional statements using `and`, `or`, `not`. |
| **Assignment Operators** | Assign and update variable values (e.g. `+=`, `-=`, `*=`). |
| **Bitwise Operators** | Perform operations on binary data (`&`, `|`, `^`, `<<`, `>>`). |
| **Membership Operators** | Check for membership in sequences using `in` or `not in`. |
| **Identity Operators** | Compare memory locations using `is` and `is not`. |
